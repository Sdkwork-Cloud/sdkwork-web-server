# SDKWork Web Server App API Route Specs

`component.spec.json` declares the executable app-api router, gateway mount, request contract port, and generated route-manifest authority.

Generated `src/http_route_manifest.rs` is not hand-edited. Route changes originate from the app OpenAPI/materialization source and must keep the SDK family aligned.

Verify from the repository root with the crate tests and SDKWork route/component validators.
