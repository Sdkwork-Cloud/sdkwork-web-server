# SDKWork Web Server Backend API Route Specs

`component.spec.json` declares the executable backend-api router, agent/operator auth boundary, service port, and generated route-manifest authority.

Generated `src/http_route_manifest.rs` is not hand-edited. Contract changes originate from the backend OpenAPI/materialization source and must keep the backend SDK aligned.

Verify from the repository root with the crate tests and SDKWork route/component validators.
